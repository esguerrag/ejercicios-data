# Curso: IA Aplicada al Análisis de Datos

Este repositorio contiene materiales y ejercicios prácticos diseñados para aprender a utilizar la Inteligencia Artificial en el análisis de datos de manera sencilla y directa.

## Estructura del repositorio

El contenido está organizado en sesiones temáticas:

```
curso-ia-datos/
├── sesion1/
│   ├── ventas_retail.csv       # Dataset de ventas por producto, región y canal
│   └── ejercicios.md           # Instrucciones de los ejercicios (Análisis Exploratorio)
├── sesion2/
│   ├── clientes_sucios.csv     # Dataset con problemas de calidad intencionales
│   └── ejercicios.md           # Instrucciones (Limpieza de Datos)
├── sesion3/
│   ├── churn_clientes.csv      # Dataset de clientes con variable objetivo churn
│   └── ejercicios.md           # Instrucciones (Modelado Predictivo)
└── sesion4/
    └── ejercicios.md           # Instrucciones (Storytelling y Visualización)
```

## Datasets incluidos

Los datasets son pequeños y fáciles de manejar, ideales para realizar pruebas rápidas con herramientas de IA:

| Dataset | Sesión | Descripción |
|---------|--------|-------------|
| `ventas_retail.csv` | 1 | 58 registros de ventas de productos tecnológicos. |
| `clientes_sucios.csv` | 2 | 35 registros con errores intencionales para practicar limpieza. |
| `churn_clientes.csv` | 3 | 50 registros de telecomunicaciones para clasificación simple. |

## Sobre los ejercicios

Los ejercicios han sido diseñados para ser **sencillos, prácticos y accesibles**:
- Se centran en el uso de herramientas de IA (como ChatGPT) para asistir en las tareas.
- No requieren conocimientos avanzados de programación.
- El objetivo es entender el flujo de trabajo y la toma de decisiones basada en datos.

*Nota: Este repositorio es de consulta personal. No es necesario realizar 'push' ni subir cambios aquí.*
